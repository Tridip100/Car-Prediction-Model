This project is a machine learning–based car price prediction system designed to estimate the resale value of used cars using various vehicle attributes. Multiple regression algorithms, including Linear Regression and Random Forest Regressor, were trained and evaluated to determine the best-performing model. Based on performance metrics such as R² score and RMSE, Random Forest Regressor was selected as the final model due to its superior predictive accuracy. The trained model is served through a RESTful API built with FastAPI, allowing users to send input data and receive real-time price predictions.
